sudo

sudo命令能让一个用户以某个用户的身份去执行命令，例如可以让普通用户以root身份查看/etc/shadow文件

sudo与直接用root身份执行的区别：
1. 使用sudo可以不需要知道root密码，可以给某用户在一段时间内拥有某些命令的root操作权限。
2. sudo可以保持当前用户的环境变量
3. sudo便于审计，系统审计日志会记录具体对应用户执行的命令
4. sudo使用时间戳文件来完成类似“检票”的系统，默认存活期为5分钟的“入场券”
sudo相关参数

1./etc/sudoers，sudo的授权文件，只有root能够编辑
visudo：专用的sudoers文本编辑工具

授权规则格式： 用户 登入主机=(代表用户) 命令
示例： root ALL=(ALL) ALL
格式说明：
user: 运行命令者的身份
host: 表示该用户可以在哪些主机上运行sudo，可以用hostname也可以用ip指定
(runas)：以哪个用户的身份
command: 允许执行和不允许执行的命令(多个命令间用逗号分隔)和特殊权限，命令可以带其选项及参数。命令要写绝对路径。不允许执行的命令需要在命令前加上”!”来表示。可以使用标签，如NOPASSWD标签表示切换或以指定用户执行该标签后的命令时不需要输入密码。一行写不下时可使用”\”续行。
